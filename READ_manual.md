# Steps to get working

### Prerequisites

- Download Ruby

```bash
gem install bundler
```
<!-- installs bundler -->

```bash
bundle install
```
<!-- installs ruby dependencies -->

### Activating Virtual Machine

1. git clone repo
<!-- to take all the code & the initialized vagrant -->

2. in root of repo folder:
```bash
vagrant up
```
<!-- to start up vagrant  -->

3. That's it!
